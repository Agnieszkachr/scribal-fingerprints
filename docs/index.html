The Typological Structure of Textual Variation

Agnieszka B. Ziemińska

A computational analysis of aggregate change profiles across five major Greek witnesses to the Gospel of John — P66, Codex Sinaiticus (ℵ), Codex Alexandrinus (A), Codex Vaticanus (B), and Codex Bezae (D) — demonstrating that each manuscript carries a statistically distinct, reproducible scribal fingerprint.

Key Findings

Primary analysis: full Gospel of John, Baseline B (Codex Vaticanus)

12,460 Token-level variants Across 21 chapters
— Cohen's d Effect size (embedding space)
— Adjusted Rand Index Clustering agreement
— Cramér's V Variant sub-type association
— Odd–Even ρ Split-half reproducibility
All reported values are derived programmatically from the pipeline output files. The signal persists across three independent baselines (B, A, and verse-matched) and survives content ablation, confirming that scribal change profiles are not artifacts of a single reference text.
