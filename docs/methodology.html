Variant Typology

Each extracted variant is classified into one of five discrete categories based on the relationship between the baseline reading and the manuscript reading:

CategoryDescriptionExample
Morphological Substitution Same lemma, different inflectional form λέγει → λέγουσιν
Lexical Substitution Different lemma replacing the original word κύριος → θεός
Addition Word present in manuscript but absent from baseline [OMIT] → δέ
Omission Word present in baseline but absent from manuscript αὐτοῦ → [OMIT]
Word Order Shift Same words in different sequence Detected at alignment level (count: 0 in current pipeline)
Morphological substitutions dominate across all manuscripts (67–83% of all variants), consistent with the expectation that most scribal changes involve minor inflectional adjustments rather than substantive lexical replacements.

Pipeline Scripts

The analysis is implemented in six Python scripts, each handling a distinct stage of the pipeline:

ScriptPurpose
download_itsee_data.py Fetches TEI XML transcriptions from ITSEE, normalises orthography, extracts and classifies variants
run_statistical_tests.py BERT embedding, permutation tests (ARI, Cohen's d), bootstrap confidence intervals, Cramér's V
create_visualizations.py PCA scatter, dendrogram, variant type distribution, cumulative signal, stability heatmap
create_verse_matched_dataset.py Identifies shared verse set across all manuscripts for controlled comparison
recalc_odd_even.py Split-half reproducibility test (Spearman ρ on manuscript distance matrices)
run_ablation_study.py Function-word filtering and content-masking ablation experiments
